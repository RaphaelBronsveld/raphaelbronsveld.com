# Raphaelbronsveld.com

Portfolio / Blog built with React Router v7 & Biome & Tailwind V4.

## Features
- MDX based blog posts with frontmatter / rehype
- Basic SEO set-up
- Always some TODO's present

## Development

Install dependencies with `npm install` 

Run the dev server:

```shellscript
npm run dev
```
